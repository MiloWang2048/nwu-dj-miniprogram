# 数据库设计

## 设计概要

- 仅设逻辑外键，无物理外键
- 使用自增主键，类型为`int(10) UNSIGNED NOT NULL AUTO_INCREMENT`
- 索引在后端开发中视情况创建
- 使用逻辑删除，用`is_deleted`标记，定义为`tinyint(1) UNSIGNED NOT NULL DEFAULT 0`
- 使用`cst_create`/`cst_modified`标记创建、修改时间，定义为`datetime(0) NOT NULL`
